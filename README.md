# DonateCare ❤️

DonateCare is a full-stack blood donation and emergency healthcare platform built using React, Vite, Express, and MongoDB.

## 🌟 Features

- 🩸 Blood Donation Management
- 🚨 Emergency Blood Request System
- 🤖 AI Chatbot Support
- 🔐 User Authentication
- 👨‍💼 Admin Dashboard
- 💳 QR Code Donation Support
- 📱 Responsive UI
- ⚡ Fast Frontend with Vite

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
